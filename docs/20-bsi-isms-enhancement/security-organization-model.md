# Security Organization Model

A functioning information security management system (ISMS) requires a defined security organization. This includes roles, responsibilities, competences, reporting paths, deputies, and communication channels.

## Core roles

| Role | Purpose |
|---|---|
| Top management | Overall accountability and decision authority |
| Information Security Officer / ISMS Manager | Coordinates and drives the security process |
| Risk owner | Accepts and owns risks |
| Asset owner | Defines protection needs for assets |
| Control owner | Ensures control design and operation |
| Process owner | Ensures business process security |
| IT/security operations | Operates technical controls |
| Internal auditor | Provides independent assurance |
| Data protection / privacy role | Advises on privacy obligations |
| Business continuity role | Coordinates resilience and recovery |

## Deputy arrangements

Important functions require deputies or substitutes.

Examples:

- ISMS manager deputy
- incident response deputy
- system owner deputy
- risk owner alternate
- emergency approval backup

## Communication paths

Define:

- who reports to whom
- when incidents are escalated
- when risk decisions go to management
- when security is involved in projects
- when suppliers notify the organization
- how policy changes are communicated

## Best practices

- Publish the security organization model internally.
- Include direct reporting path from the security officer to top management.
- Define when the security officer must be consulted.
- Review structure after reorganizations, mergers, or major outsourcing.
- Ensure roles have resources and authority.

## Evidence

- security organization chart
- responsible, accountable, consulted, and informed (RACI) matrix
- role descriptions
- deputy list
- communication matrix
- governance meeting minutes


## Practical example

An organization uses this guidance to strengthen an existing ISO/IEC 27001 process with a clearly assigned security role, direct escalation, and practical operating records suited to its governance model.
