# External Information Security Officer Contract Checklist

When an organization uses an external information security officer, the contract must clearly define duties, rights, confidentiality, reporting, and transition.

## Contract topics

- scope of services
- duties and deliverables
- reporting obligations
- access rights
- confidentiality / NDA
- conflict-of-interest rules
- availability and response times
- participation in projects
- participation in management review
- incident escalation duties
- documentation ownership
- handover at termination
- data protection requirements
- subcontractor restrictions
- liability and insurance
- termination and transition support

## Operational checklist

- [ ] Role and tasks defined
- [ ] Reporting line to management defined
- [ ] Confidentiality agreement included
- [ ] Access rights approved
- [ ] Conflict-of-interest reviewed
- [ ] Deliverables defined
- [ ] Handover obligations defined
- [ ] Exit process defined
- [ ] Backup contact defined
- [ ] Performance review scheduled

## Evidence

- signed contract
- NDA
- role description
- access approval
- deliverable records
- meeting minutes
- handover records
