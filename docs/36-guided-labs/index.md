# Guided information security management system (ISMS) Labs

These labs are designed for self-study, workshops, team training, and auditor preparation. Each lab includes a realistic scenario, learning objectives, supplied facts, tasks, expected deliverables, review questions, and a suggested answer structure.

The suggested answers are not the only acceptable answers. The purpose is to practice structured reasoning, evidence design, and cross-linking between ISMS documents.

## How to use this section

Start with the overview of **Guided ISMS Labs**, then follow the linked articles according to the decision or task at hand. Use the related templates to record decisions and the checklists to verify completion. Each linked article distinguishes formal ISO requirements from implementation guidance and optional best practice.
