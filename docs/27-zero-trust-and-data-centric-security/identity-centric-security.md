# Identity-Centric Security

Identity is the foundation of Zero Trust.

## Key capabilities

- unique user identities
- strong authentication
- MFA for sensitive access
- service identity governance
- privileged access management
- access reviews
- joiner/mover/leaver integration
- conditional access policies
- identity threat detection

## Evidence examples

- MFA coverage report
- privileged account list
- access review record
- conditional access policy export
- service account owner list
- identity incident records


## Typical evidence

- approved policy, standard, procedure, or architecture record
- risk assessment or design review
- owner and role assignment
- implementation plan
- operating records
- monitoring records
- exception or waiver decisions
- test results
- audit records
- management review decisions
