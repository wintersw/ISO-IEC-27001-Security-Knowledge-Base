# Security Organization Review Checklist

- [ ] roles defined
- [ ] owners assigned
- [ ] competences sufficient
- [ ] resources sufficient
- [ ] deputies defined
- [ ] communication paths documented
- [ ] direct reporting path available
- [ ] structure reviewed after changes
