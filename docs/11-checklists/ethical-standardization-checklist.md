# Ethical Standardization Checklist

- [ ] purpose is clear
- [ ] affected stakeholders identified
- [ ] proportionality assessed
- [ ] privacy impact considered
- [ ] participation opportunity provided
- [ ] exceptions process defined
- [ ] sanctions are fair
- [ ] communication plan prepared
- [ ] review date defined

## Evidence

Retain the completed checklist, reviewer, date, decisions, and follow-up actions.
