# Data Security Lifecycle Checklist

- [ ] collection purpose defined
- [ ] classification assigned
- [ ] owner assigned
- [ ] storage approved
- [ ] access rules defined
- [ ] sharing approved
- [ ] retention defined
- [ ] deletion process defined
- [ ] logging enabled
- [ ] breach impact documented

Retain the completed checklist with reviewer, date, decision, and follow-up actions.
