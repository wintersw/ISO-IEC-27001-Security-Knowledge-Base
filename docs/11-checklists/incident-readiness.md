# Incident Readiness

Use this checklist with the incident manager, service owners, legal or privacy specialists, and communications leads. Validate each item through records or a tabletop exercise rather than relying only on documented intent.

- [ ] procedure approved
- [ ] roles defined
- [ ] contacts current
- [ ] severity model defined
- [ ] reporting channels communicated
- [ ] tabletop performed
- [ ] lessons learned tracked
