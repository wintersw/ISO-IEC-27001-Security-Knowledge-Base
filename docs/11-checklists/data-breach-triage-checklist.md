# Data Breach Triage Checklist

- [ ] incident confirmed
- [ ] data affected identified
- [ ] classification checked
- [ ] affected parties identified
- [ ] containment started
- [ ] evidence preserved
- [ ] legal/privacy escalation completed
- [ ] notification decision recorded
- [ ] management informed

## Evidence

Retain the completed checklist with reviewer, date, decision, and follow-up actions.
