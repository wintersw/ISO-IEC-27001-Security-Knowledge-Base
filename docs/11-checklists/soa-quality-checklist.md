# Statement of Applicability Quality Checklist

- [ ] all Annex A controls are represented
- [ ] applicability is marked for each control
- [ ] inclusion justifications are risk-based or requirement-based
- [ ] exclusion justifications are documented
- [ ] implementation status is current
- [ ] control owners are assigned
- [ ] evidence references are included
- [ ] related risks are linked
- [ ] SoA is approved
- [ ] SoA is reviewed after risk or scope changes

## Evidence

Retain the completed checklist with reviewer, date, decision, and follow-up actions.
