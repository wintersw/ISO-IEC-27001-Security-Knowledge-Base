# Zero Trust Readiness Checklist

- [ ] protect surfaces identified
- [ ] identity controls reviewed
- [ ] device posture reviewed
- [ ] network segmentation reviewed
- [ ] application access model reviewed
- [ ] data access controls reviewed
- [ ] monitoring integrated
- [ ] exceptions managed
