# Privacy Engineering Review Checklist

- [ ] personal data identified
- [ ] purpose documented
- [ ] minimization performed
- [ ] retention defined
- [ ] data flow mapped
- [ ] access controlled
- [ ] masking or pseudonymization considered
- [ ] privacy testing planned
- [ ] evidence defined

## Evidence

Retain the completed checklist with reviewer, date, decision, and follow-up actions.
