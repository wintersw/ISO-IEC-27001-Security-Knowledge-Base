# Implementation Roadmap Quality Checklist

- [ ] management support confirmed
- [ ] scope defined
- [ ] context and interested parties identified
- [ ] assets and protection needs assessed
- [ ] risk methodology approved
- [ ] risk assessment completed
- [ ] SoA created
- [ ] controls prioritized
- [ ] evidence model defined
- [ ] internal audit planned
- [ ] management review planned
- [ ] certification readiness assessed
