# Security Lifecycle Review Checklist

- [ ] lifecycle item has an owner
- [ ] current lifecycle stage is known
- [ ] related risks are linked
- [ ] related controls are linked
- [ ] evidence is defined
- [ ] monitoring is defined
- [ ] assurance method is defined
- [ ] retirement criteria are defined
- [ ] improvement actions are tracked

## Evidence

Retain the completed checklist with reviewer, date, decisions, and follow-up actions.
