# Corrective Action Effectiveness Checklist

- [ ] original issue understood
- [ ] root cause documented
- [ ] implemented action matches root cause
- [ ] evidence reviewed
- [ ] sample or metric selected
- [ ] recurrence checked
- [ ] affected similar areas considered
- [ ] risk register updated if needed
- [ ] SoA updated if needed
- [ ] closure decision documented

## Evidence

Retain the completed checklist with reviewer, date, decision, and follow-up actions.
