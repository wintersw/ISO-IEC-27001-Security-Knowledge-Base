# Security Resource Economics Checklist

- [ ] risk addressed identified
- [ ] business benefit defined
- [ ] implementation cost estimated
- [ ] operating cost estimated
- [ ] owner assigned
- [ ] priority justified
- [ ] management decision documented
- [ ] effectiveness review planned
