# Improvement Backlog Review Checklist

- [ ] new items reviewed
- [ ] duplicate items merged
- [ ] priority confirmed
- [ ] owner assigned
- [ ] due date realistic
- [ ] resource blockers identified
- [ ] management decisions identified
- [ ] completed items have evidence
- [ ] effectiveness review scheduled
- [ ] overdue items escalated

## Evidence

Retain the completed checklist with reviewer, date, decision, and follow-up actions.
