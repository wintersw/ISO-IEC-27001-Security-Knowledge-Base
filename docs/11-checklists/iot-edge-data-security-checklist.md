# Iot Edge Data Security Checklist

- [ ] device identity defined
- [ ] telemetry classified
- [ ] firmware update controlled
- [ ] transport encrypted
- [ ] edge processing documented
- [ ] supplier role reviewed
- [ ] monitoring defined
- [ ] decommissioning process defined
