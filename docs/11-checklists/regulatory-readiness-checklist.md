# Regulatory Readiness Checklist

- [ ] obligation source identified
- [ ] applicability assessed
- [ ] requirements mapped to ISO clauses/controls
- [ ] owners assigned
- [ ] evidence identified
- [ ] gaps recorded
- [ ] actions tracked
- [ ] management informed
- [ ] review cycle established
