# Service Design Security Checklist

- [ ] service owner assigned
- [ ] data classification defined
- [ ] risk assessment completed
- [ ] access model defined
- [ ] logging requirements defined
- [ ] backup/recovery requirements defined
- [ ] supplier dependencies reviewed
- [ ] security testing planned
- [ ] incident escalation defined

## Evidence

Retain the completed checklist with reviewer, date, decisions, and follow-up actions.
