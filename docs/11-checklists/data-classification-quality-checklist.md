# Data Classification Quality Checklist

- [ ] classification criteria applied
- [ ] owner approved classification
- [ ] privacy impact considered
- [ ] legal/contractual obligations considered
- [ ] handling rules linked
- [ ] labeling implemented
- [ ] exceptions documented
- [ ] review date defined

## Evidence

Retain the completed checklist with reviewer, date, decision, and follow-up actions.
