# Employee Security Onboarding Checklist

- [ ] role approved
- [ ] NDA or confidentiality obligations completed
- [ ] background screening completed where applicable
- [ ] asset issued and recorded
- [ ] MFA enabled
- [ ] least-privilege access approved
- [ ] security awareness assigned
- [ ] manager confirms access need

## Evidence

Retain the completed checklist or workflow record with date, owner, decisions, and follow-up actions.
