# Perceived Participation Rollout Checklist

- [ ] affected groups identified
- [ ] risk and purpose explained
- [ ] pilot group selected
- [ ] feedback collected
- [ ] feedback considered
- [ ] changes communicated
- [ ] support channel defined
- [ ] post-rollout survey completed

## Evidence

Retain the completed checklist, reviewer, date, decisions, and follow-up actions.
