# Template Tailoring Checklist

- [ ] template copied into working area
- [ ] organization name and scope updated
- [ ] owners and approvers assigned
- [ ] generic examples removed or marked as examples
- [ ] legal, contractual, and customer requirements considered
- [ ] links to related risks, controls, and evidence added
- [ ] review frequency defined
- [ ] retention requirement defined
- [ ] document classification defined
- [ ] approval record captured

## Evidence

Retain the completed checklist with reviewer, date, decision, and follow-up actions.
