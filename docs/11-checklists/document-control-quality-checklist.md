# Document Control Quality Checklist

- [ ] document ID exists
- [ ] version is controlled
- [ ] owner is assigned
- [ ] approval authority is recorded
- [ ] current version is published
- [ ] obsolete versions are archived
- [ ] review date is tracked
- [ ] classification is assigned
- [ ] retention is defined
- [ ] access permissions are appropriate

## Evidence

Retain the completed checklist with reviewer, date, decision, and follow-up actions.
