# Certification Readiness

Complete this high-level check before committing to an audit date. A checked item should be supported by approved, current evidence; use the linked stage-readiness checklists for a deeper review and assign an owner and due date for every gap.

- [ ] ISMS scope approved
- [ ] interested parties identified
- [ ] climate relevance considered
- [ ] policy approved
- [ ] risk assessment completed
- [ ] SoA current
- [ ] internal audit completed
- [ ] management review completed
- [ ] corrective actions tracked
