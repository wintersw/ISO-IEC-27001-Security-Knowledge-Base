# Supplier Security Review Checklist

- [ ] supplier inventory updated
- [ ] risk rating confirmed
- [ ] security questionnaire reviewed
- [ ] contract clauses verified
- [ ] subprocessors reviewed
- [ ] incidents or changes reviewed
- [ ] residual risk approved

## Evidence

Retain the completed checklist or workflow record with date, owner, decisions, and follow-up actions.
