# Surveillance Audit Preparation Checklist

- [ ] review previous audit findings
- [ ] update risk register
- [ ] confirm SoA accuracy
- [ ] collect recent operating evidence
- [ ] review management review actions
- [ ] test selected controls
- [ ] prepare control owners
- [ ] confirm corrective action status
