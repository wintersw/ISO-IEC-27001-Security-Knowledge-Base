# Framework Selection Checklist

- [ ] business objective defined
- [ ] assurance need identified
- [ ] certification need confirmed
- [ ] scope defined
- [ ] overlap with existing frameworks assessed
- [ ] implementation resources estimated
- [ ] evidence reuse identified
- [ ] decision documented

## Evidence

Retain the completed checklist, reviewer, date, decisions, and follow-up actions.
