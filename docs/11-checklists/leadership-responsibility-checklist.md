# Leadership Responsibility Checklist

- [ ] overall responsibility formally accepted
- [ ] security process initiated
- [ ] security objectives approved
- [ ] security strategy approved
- [ ] roles assigned
- [ ] resources allocated
- [ ] ISMS status reviewed regularly
- [ ] risks and consequences understood
- [ ] decisions documented

## Evidence

Retain the completed checklist, decisions, approvals, and follow-up actions.
