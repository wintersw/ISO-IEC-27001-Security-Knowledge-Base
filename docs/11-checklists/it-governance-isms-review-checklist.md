# IT Governance ISMS Review Checklist

- [ ] security objectives linked to business objectives
- [ ] major IT risks visible to governance
- [ ] residual risk decisions documented
- [ ] resource constraints escalated
- [ ] security metrics reviewed
- [ ] supplier risk reviewed
- [ ] technical debt reviewed
- [ ] management decisions tracked

## Evidence

Retain the completed checklist with reviewer, date, decisions, and follow-up actions.
