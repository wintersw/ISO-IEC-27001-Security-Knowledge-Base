# Control Assurance Review Checklist

- [ ] control objective defined
- [ ] control owner confirmed
- [ ] population defined
- [ ] sample selected
- [ ] design effectiveness tested
- [ ] operating effectiveness tested
- [ ] exceptions reviewed
- [ ] evidence quality assessed
- [ ] findings recorded
- [ ] actions assigned
