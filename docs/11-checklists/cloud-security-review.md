# Cloud Security Review Checklist

- [ ] cloud assets inventoried
- [ ] privileged access reviewed
- [ ] MFA enforced
- [ ] logging enabled
- [ ] public exposure checked
- [ ] backup and recovery checked
- [ ] configuration findings remediated
- [ ] exceptions approved
