# External ISB Contract Checklist

- [ ] tasks defined
- [ ] rights and duties defined
- [ ] confidentiality agreement included
- [ ] direct reporting path defined
- [ ] project involvement defined
- [ ] handover obligations included
- [ ] termination process defined
- [ ] access rights approved
