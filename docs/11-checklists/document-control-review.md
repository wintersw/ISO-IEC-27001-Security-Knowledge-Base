# Document Control Review

Use this checklist during scheduled reviews and after material process, legal, organizational, or technology changes. Record the document reviewed, reviewer, date, findings, actions, and next review date so the completed checklist can serve as evidence.

- [ ] owner assigned
- [ ] approver recorded
- [ ] version current
- [ ] scope accurate
- [ ] review date not overdue
- [ ] requirements match practice
- [ ] superseded versions controlled
