# Greenfield ISO 27001 Roadmap

## Phase 1 — Prepare

- appoint sponsor
- appoint ISMS lead
- define objectives
- confirm resources
- select documentation platform
- train core team

## Phase 2 — Define scope and context

- identify services and business processes
- define scope
- identify interested parties
- document requirements
- consider climate-related relevance
- map dependencies

## Phase 3 — Build risk process

- define methodology
- create asset inventory
- identify risks
- assess and evaluate risks
- approve treatment approach

## Phase 4 — Implement controls

- create SoA
- assign control owners
- implement priority controls
- retain evidence
- track exceptions

## Phase 5 — Evaluate and improve

- perform internal audit
- conduct management review
- close corrective actions
- prepare for certification

## Success factors

- executive sponsorship
- realistic scope
- clear owners
- evidence by design
- risk-based prioritization
