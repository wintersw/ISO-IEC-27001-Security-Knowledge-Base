# Implementation Guides

Different organizations need different implementation patterns. A startup, enterprise, software as a service (SaaS) provider, manufacturer, and public-sector body may all implement ISO/IEC 27001 differently while meeting the same management-system principles.

## How to use this section

Start with the overview of **Implementation Guides**, then follow the linked articles according to the decision or task at hand. Use the related templates to record decisions and the checklists to verify completion. Each linked article distinguishes formal ISO requirements from implementation guidance and optional best practice.
