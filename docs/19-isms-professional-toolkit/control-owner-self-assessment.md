# Control Owner Self-Assessment

Control owner self-assessments help maintain visibility between formal audits.

## Purpose

The assessment asks control owners to confirm whether their controls are designed, operating, evidenced, and improving.

## Self-assessment questions

| Question | Response |
|---|---|
| Is the control still applicable? | Yes / No / Unsure |
| Is the control owner still correct? | Yes / No |
| Has the control operated during the period? | Yes / No / Partially |
| Is required evidence available? | Yes / No / Partially |
| Were exceptions identified? | Yes / No |
| Were exceptions approved? | Yes / No / N/A |
| Were remediation actions completed on time? | Yes / No / N/A |
| Did any incidents indicate control weakness? | Yes / No |
| Did any changes affect the control? | Yes / No |
| Is the SoA status still accurate? | Yes / No |

## Rating model

| Rating | Meaning |
|---|---|
| Effective | Control is operating and evidenced |
| Partially effective | Control operates with gaps or incomplete evidence |
| Ineffective | Control failed or is not operating |
| Not assessed | Insufficient information |
| Not applicable | Control no longer applies |

## Best practices

- Require evidence references, not only yes/no responses.
- Challenge “effective” ratings where incidents or findings exist.
- Use self-assessments to prioritize assurance testing.
- Track repeat partial ratings as improvement opportunities.

## Evidence

- completed self-assessment
- supporting records
- exception register
- remediation tracker
- control assurance results


## Practical example

Ahead of the semi-annual assurance cycle, the backup control owner completes the self-assessment and rates the control "partially effective": restore tests ran on schedule, but two months of test evidence are missing. The ISMS team challenges an "effective" rating submitted by the vulnerability management owner because a recent incident indicated scanning gaps, and both controls are prioritized for independent assurance testing in the next audit.
