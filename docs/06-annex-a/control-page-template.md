# Annex A Control Page Template

Use this structure for every control page.

```markdown
# A.x.x Control Name

## Overview
## Purpose
## Risks addressed
## Key concepts
## Practical implementation
## Small organization example
## Enterprise example
## Cloud / SaaS considerations
## Roles and responsibilities
## Evidence
## Metrics and KPIs
## Common mistakes
## Auditor questions
## Checklist
## Related controls
```
