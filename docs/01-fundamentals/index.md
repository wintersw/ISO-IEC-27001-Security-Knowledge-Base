# Information Security Fundamentals

Information security protects information and the systems, people, facilities, and services that create, process, store, transmit, or depend on it.

A mature security program protects business outcomes rather than technologies in isolation. A server matters because it supports a service; the service matters because a business process and interested parties depend on it.
