# Improvement Prioritization Matrix Template

| Improvement ID | Risk reduction 1–5 | Urgency 1–5 | Compliance driver 1–5 | Business value 1–5 | Recurrence prevention 1–5 | Evidence value 1–5 | Effort 1–5 | Priority score | Priority |
|---|---:|---:|---:|---:|---:|---:|---:|---:|---|
| IMP-001 |  |  |  |  |  |  |  |  | P1 / P2 / P3 / P4 |

## Suggested formula

```text
Priority score = risk reduction + urgency + compliance driver + business value + recurrence prevention + evidence value - effort
```

## Priority interpretation

| Priority | Interpretation |
|---|---|
| P1 | urgent / high-risk / management attention |
| P2 | important in current planning cycle |
| P3 | useful when resources allow |
| P4 | monitor, defer, or combine with other work |

## Usage guidance

Use this template to document **Improvement Prioritization Matrix**.

## Evidence to retain

Retain the approved record, source evidence, approval history, exceptions, and follow-up actions under the organization's retention and protection rules.

## Practical example

For one in-scope service, the owner completes the **Improvement Prioritization Matrix** record, links authoritative evidence, obtains review, and tracks rejected or incomplete items to closure.

## ISO requirement, implementation guidance, and best practice

This exact template is not an ISO requirement; it is guidance for recording **Improvement Prioritization Matrix** consistently. Controlled values, named owners, timestamps, approvals, and authoritative evidence improve assurance.
