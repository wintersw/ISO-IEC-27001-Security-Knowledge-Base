# Perceived Participation Survey Template

Use a 1–5 scale.

| Statement | Score |
|---|---|
| I understood why this security change was introduced. |  |
| I had an opportunity to provide feedback. |  |
| My team’s operational needs were considered. |  |
| The final process is practical for my work. |  |
| I know how to request help or an exception. |  |
| Communication before rollout was sufficient. |  |
| The security change improves protection without unnecessary friction. |  |

## Free-text

- What should be improved?
- What risk or workaround do you see?
- What additional guidance would help?
