# Security Culture Survey Template

Use a 1–5 scale where 1 means strongly disagree and 5 means strongly agree.

| Statement | Score |
|---|---|
| I know how to report a suspected security incident. |  |
| I understand my responsibility for protecting company information. |  |
| I know where to find security policies. |  |
| I feel comfortable reporting a mistake or security concern. |  |
| Security requirements are clear in my role. |  |
| Security processes help me work safely. |  |
| I have received enough training for the information I handle. |  |

## Free-text questions

- What security process is hardest to follow?
- What security topic do you need more training on?
- What would make secure behavior easier?
