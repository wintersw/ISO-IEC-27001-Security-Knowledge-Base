# Management Review Minutes Template

Prepare input material before the meeting and record enough context to show what top management considered. The output should make decisions explicit, including changes to the ISMS, resource commitments, accepted escalations, and improvement actions.

## Inputs reviewed

- previous actions
- internal/external changes
- interested-party changes
- climate relevance where applicable
- risk and treatment status
- objective performance
- audit results
- incidents
- nonconformities
- metrics
- resource needs
- improvement opportunities
