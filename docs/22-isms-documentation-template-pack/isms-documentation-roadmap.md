# ISMS Documentation Roadmap
This roadmap shows the recommended order for creating and applying ISMS documents.

## Phase 1 — Governance foundation

Create:

1. ISMS scope statement
2. ISMS manual or overview
3. Information security policy
4. Roles, responsibilities, and responsible, accountable, consulted, and informed (RACI)
5. Document control procedure
6. Interested-party and requirements register

## Phase 2 — Risk and applicability

Create:

1. Asset inventory
2. Protection-needs analysis
3. Risk assessment methodology
4. Risk assessment worksheet
5. Risk register
6. Risk treatment plan
7. Risk acceptance record
8. Statement of Applicability

## Phase 3 — Control implementation

Create:

1. Control implementation plan
2. Control owner matrix
3. Evidence register
4. Exception register
5. Operating procedures
6. Access review records
7. Supplier assessments
8. Incident response records

## Phase 4 — Assurance and evaluation

Create:

1. Internal audit program
2. Internal audit plan
3. Audit checklist
4. Audit report
5. Nonconformity report
6. Corrective action plan
7. Control test records
8. key performance indicator (KPI)/key risk indicator (KRI) dashboard

## Phase 5 — Management review and improvement

Create:

1. Management review pack
2. Management review minutes
3. Management decision log
4. Improvement backlog
5. Lessons learned records
6. Continual improvement tracker

## Practical sequencing

Do not start by writing every policy. Start by confirming scope, risk methodology, asset ownership, and the minimum policy framework. Then build controls and evidence around real risks.

## Practical example

An ISMS coordinator uses this guidance to select the minimum useful document, assigns an owner and approver, and connects the controlled document to the process records that prove actual operation.

## Evidence to retain

Retain records showing both design decisions and actual operation, such as:

- document owner and approval
- version and change history
- distribution or acknowledgement record
- linked operating evidence
