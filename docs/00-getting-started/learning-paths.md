# Learning Paths

## Security engineer

Fundamentals → risk management → technological controls → evidence by design → operational checklists.

## ISO implementer

ISO overview → clauses → ISMS → risk → Annex A → documentation → internal audit.

## Internal auditor

Clauses → Annex A → evidence → audit process → nonconformities → corrective actions.

## Security leader

Why ISO 27001 → governance → leadership → risk appetite → management review → KPIs.
