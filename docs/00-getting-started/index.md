# Getting Started

ISO/IEC 27001 is a management-system standard. It does not prescribe a fixed technology stack or require every organization to implement identical controls. It requires an organization to understand its context, assess information security risk, select and operate appropriate treatment, evaluate performance, and improve.

## What this knowledge base is

- A learning resource for new security-team members.
- An implementation guide for ISMS and control owners.
- An operational reference for security engineering and GRC.
- A source of templates, checklists, examples, and diagrams.

## What it is not

- A replacement for the official standard.
- Legal advice.
- A guarantee of certification.
- A generic checklist that removes the need for risk-based decisions.
