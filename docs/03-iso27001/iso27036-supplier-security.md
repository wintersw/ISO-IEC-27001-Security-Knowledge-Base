# ISO/IEC 27036 Supplier Relationship Security

ISO/IEC 27036 provides guidance for information security in supplier relationships.

## Why it matters

Modern organizations depend on SaaS providers, cloud platforms, managed service providers, contractors, and outsourced development teams. Supplier weaknesses can become organizational incidents.

## Practical supplier lifecycle

1. Identify supplier need.
2. Classify supplier risk.
3. Perform due diligence.
4. Include security requirements in contract.
5. Onboard securely.
6. Monitor performance.
7. Review changes.
8. Offboard and confirm data return or deletion.

## Evidence

- supplier inventory
- supplier risk rating
- due diligence assessment
- contract security clauses
- monitoring records
- offboarding evidence
