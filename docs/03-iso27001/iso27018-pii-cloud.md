# ISO/IEC 27018 and PII in Public Cloud

ISO/IEC 27018 provides guidance for protecting personally identifiable information in public cloud environments where the cloud provider acts as a processor.

## Key themes

- processor obligations
- customer instructions
- transparency
- access control
- data deletion and return
- subprocessor controls
- disclosure handling
- audit and compliance

## Practical use

Security and privacy teams should use it to strengthen cloud privacy controls, especially for SaaS and public cloud services that process customer or employee data.

## Evidence

- data processing agreements
- supplier assessment
- subprocessor list
- deletion/return records
- access-control evidence
- privacy impact assessment
