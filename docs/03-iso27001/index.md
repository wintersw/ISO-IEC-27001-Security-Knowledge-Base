# ISO/IEC 27001 Overview

ISO/IEC 27001 defines requirements for an Information Security Management System. Clauses 4–10 contain the certifiable management-system requirements. Annex A provides a reference set of 93 information security controls organized into organizational, people, physical, and technological themes.

## A key distinction

The organization must satisfy applicable clauses 4–10 within the ISMS scope. Annex A controls are not a universal checklist. Control selection must follow risk treatment and must be explained in the Statement of Applicability.

## ISO/IEC 27001 and ISO/IEC 27002

ISO/IEC 27001 provides ISMS requirements. ISO/IEC 27002 provides implementation guidance for information security controls. Certification is against ISO/IEC 27001, not ISO/IEC 27002.
