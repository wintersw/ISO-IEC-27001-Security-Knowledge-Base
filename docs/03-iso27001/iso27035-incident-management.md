# ISO/IEC 27035 Incident Management

ISO/IEC 27035 provides guidance for managing information security incidents. It supports preparation, detection, assessment, response, lessons learned, and improvement.

## Practical use

Use it to design:

- incident classification
- severity model
- escalation paths
- response playbooks
- evidence handling
- communication processes
- lessons learned

## Evidence

- incident response procedure
- incident tickets
- timelines
- decision logs
- evidence collection records
- post-incident review
- corrective actions
