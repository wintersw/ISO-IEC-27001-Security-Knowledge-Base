# ISO 27001 and NIST SP 800-53

NIST SP 800-53 Rev. 5 is a catalog of security and privacy controls organized into control families (for example, Access Control, Audit and Accountability, and Incident Response). Release 5.2.0 supplemental materials include NIST's official ISO/IEC 27001:2022 crosswalk. ISO/IEC 27001 defines a certifiable management system with the Annex A control reference set; mapped evidence still needs matching scope, population, period, and test purpose.

## Practical relationship

| NIST SP 800-53 family (examples) | ISO/IEC 27001 relationship | Example evidence |
|---|---|---|
| AC — Access Control | A.5.15–A.5.18, A.8.2–A.8.5 | access policy, joiner/mover/leaver records, access reviews |
| AU — Audit and Accountability | A.8.15, A.8.16 | logging standard, log coverage report, alert tests |
| CA — Assessment, Authorization, Monitoring | Clauses 9.1–9.3, A.5.35, A.5.36 | internal audit, management review, control assessments |
| CP — Contingency Planning | A.5.29, A.5.30, A.8.13, A.8.14 | continuity plans, backup and recovery tests |
| IR — Incident Response | A.5.24–A.5.28 | incident procedure, case records, lessons learned |
| RA — Risk Assessment | Clause 6.1, A.8.8 | risk methodology, risk register, vulnerability management |
| SC — System and Communications Protection | A.8.20–A.8.26 | network segmentation, cryptography, secure channels |
| SI — System and Information Integrity | A.8.7, A.8.8, A.8.16 | malware protection, patching, monitoring |

## Best practice

Treat ISO/IEC 27001 as the management-system backbone and SP 800-53 as a detailed control catalog. A control family is broader than a single Annex A control, so map at the individual control (and control enhancement) level for an operational crosswalk. NIST publishes an official ISO/IEC 27001 mapping in the SP 800-53 supplemental materials — use it as the authoritative source rather than reproducing a full crosswalk here.

## Practical example

A federal contractor already assessed against SP 800-53 moderate baseline reuses its access-review and logging evidence for ISO/IEC 27001 A.5.18 and A.8.15. Where SP 800-53 requires control enhancements without an Annex A counterpart, the team records the obligation in its own control set and Statement of Applicability.
