# Framework Mappings

Mappings help organizations reuse evidence and understand how different frameworks relate. They should not be treated as automatic compliance.

## Included mappings

- NIST Cybersecurity Framework
- CIS Controls
- SOC 2
- General Data Protection Regulation (GDPR)
- ISO 22301
- OWASP

## How to use this section

Start with the overview of **Framework Mappings**, then follow the linked articles according to the decision or task at hand. Use the related templates to record decisions and the checklists to verify completion. Each linked article distinguishes formal ISO requirements from implementation guidance and optional best practice.
