# Information Security Management System

An ISMS is the coordinated set of governance, policies, processes, responsibilities, resources, risk decisions, controls, records, and improvement activities used to manage information security.

It is not a folder of policies or a certification project. A functioning ISMS influences decisions, assigns accountability, produces evidence, and adapts to changes in business, technology, threats, obligations, and performance.

## How to use this section

Start with the overview of **Information Security Management System**, then follow the linked articles according to the decision or task at hand. Use the related templates to record decisions and the checklists to verify completion. Each linked article distinguishes formal ISO requirements from implementation guidance and optional best practice.
